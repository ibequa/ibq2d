# ibQ2D<br>

<i>open-source game development framework</i> <p>Currently has: </p> 
- input system (mouse/keyboard)
- collision detection (box/circle/edge/polygon)
- game state callbacks
- basic geometry
- texture loading
- text support, custom fonts
- scene system
- 2D animation
- basic sound
- additive scene loading
